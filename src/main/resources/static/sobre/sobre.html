Este Projeto

O KRONOS CORE é um orquestrador industrial para legendas de animes: análise de mídia, extração ASS/SRT/PGS, tradução via LLM local (Mistral), correção de cache, revisão de concordância, correção de legendas, revisão de lore, remux e telemetria — tudo em Java/Quarkus com interface web integrada.

Repositório público: carmipa/traducao_animes_llm_local_quarkus · branch main · linguagem Java
